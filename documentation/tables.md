# Tables Used in Healthcare Dashboard

## cities Table
- **Columns**:
  - City
  - City ID
  - State

## departments Table
- **Columns**:
  - Department
  - Department ID


## diagnoses Table
- **Columns**:
  - Diagnosis
  - Diagnosis ID


## insurance Table
- **Columns**:
  - Insurance ID
  - Insurance Provider


## patients Table
- **Columns**:
  - Age
  - City ID
  - Gender
  - Patient ID
  - Patient Name
  - Race


## procedures Table
- **Columns**:
  - Procedure
  - Procedure ID


## providers Table
- **Columns**:
  - Age
  - Gender
  - Image
  - Nationality
  - Provider ID
  - Provider Name

  - Insurance ID

  - Patient ID
  - Patient Satisfaction Score
  - Payment Status
  - Procedure ID
  - Provider ID
  - Referral Source
  - Room Charges (daily rate)
  - Room Type
  - Service Type
  - Treatment Cost
